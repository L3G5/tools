# RSVP Speed Reader

A Rapid Serial Visual Presentation (RSVP) speed reading tool that displays words one at a time with the optimal recognition point highlighted in red to improve reading speed and focus.

Features:
- Upload and process PDF files with automatic text extraction
- OCR support for scanned PDFs using Tesseract.js
- Click on any paragraph in the preview to select your starting point
- Adjustable reading speed (100-1000 WPM)
- Multiple speed regimes: constant, ramp-up, or custom
- Keyboard shortcuts for easy control
